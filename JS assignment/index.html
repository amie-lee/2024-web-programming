GOAL:
Create a continuous circular railway line that covers every accessible tile on the grid.

HOW TO PLAY:
  • Drag a rail from the side.
  • Drop it onto the map.

    • RULES:
  • Bridge: Only straight rails
  • Mountain: Only curved rails
  • Oasis: No rails can be placed
  • Empty: Any rail can be placed

    • * Click a rail on the map to rotate it.
